## 📦 O que tem dentro?

| Skill | O que faz |
|-------|-----------|
| `frontend-architect-lead` | Arquiteto front-end para times de 400+ devs. Performance, acessibilidade, design system. |
| `backend-architect-lead` | Arquiteto back-end para sistemas gigantes. APIs, bancos, observabilidade, segurança. |
| `token-saver` | Respostas super curtas e diretas. Economiza tokens quando você tá com pressa. |
| `bug-hunter` | Caça bugs igual detetive. Responde em 4 linhas, sem enrolação. |
| `refactor-planner` | Plano cirúrgico pra refatorar código feio em código bonito. |
| `negociador-alfa` | Te ajuda a negociar preço, salário, prazo. Baseado em método Harvard (sério). |
| `closer-de-sites` | Vende sites, landing pages, manutenção. Prospecção, objeções, fechamento. |

## 🚀 Como usar

1. Entre na pasta da skill que você quer (ex.: `frontend-architect-lead/`)
2. Copie o conteúdo do arquivo `SKILL.md`
3. Cole no campo de instruções personalizadas do seu assistente (ChatGPT, Claude, etc.)
4. Ative pelo nome ou descrição
