## 📦 O que tem dentro?

| Skill | O que faz |

| `token-saver` | Respostas super curtas e diretas. Economiza tokens. |

| `refactor-planner` | Plano cirúrgico pra refatorar código. |
| `negociador-alfa` | Negociação de preço, salário, prazo. |
| `closer-de-sites` | Venda de sites e prospecção de clientes. |
| `pro-animation-studio` | Animações profissionais prontas. Scroll video, mouse trail. |
| `wolf-of-sales` | Persona Lobo de Wall Street para vendas agressivas e fechamento de negócios. |

## 🚀 Como usar

1. Entre na pasta da skill.
2. Copie o conteúdo do arquivo `SKILL.md`.
3. Cole no campo de instruções personalizadas do seu assistente (Cursor, Copilot, etc.).
4. Ative pelo nome ou descrição.
